# Keywords

## Python

### Patterns

- `"string".function()`
- `list = [1, 2, 3]`
- `len("string")` not `"string".len()`
- assignment: `a, b = 0, 1` and `a, b = b, a+b`
- `while b < 10:`
- `print b,` avoids new-line
- `for w in words:`, indent lines after that

### Functions

- append()
- encode()
- int()
- len()
- print()
- range()
- round()
- str()
- unicode()

### Keywords

- if, elif, else
- while
